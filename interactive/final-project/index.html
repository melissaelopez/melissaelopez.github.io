Final Project: Aura Reader - Documentation

Project Proposal:

We’d like to create an experience where the world is created based on the user’s aura. The way we’d be interested in making this with a Kinect, where the experience alters the displayed world based on the user’s “aura (e.g. depending on how fast they are moving.) For example, if the user is moving really quickly, it senses a RED aura, and enters a red visual space with things that move according to the users motions. If the user moves slowly and fluidly, it senses a GREEN aura, which has a more serene world.

We have both been inspired by the game Monument Valley, so we would love to emulate this aesthetic in creating the aura-based worlds, particularly in colors and textures.

Rather than recreating the physics and functions of the game, we are using it's graphics for inspiration to create our "aura worlds". Once the program "senses" the user's aura and enters him or her into a world, the user will be able to alter and interact with aspects of the graphical world. I.e. swiping your arm fans infinite diamonds from the visual field, which slowly shrink to nothing.
