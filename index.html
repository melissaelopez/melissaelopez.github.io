BITE

Social media users need a better way to organize their timelines/feeds across multiple networks. This prototyping project aims to solve this problem with a design for creating custom user feeds based on groups of people user's would like to follow on Facebook, Twitter and Instagram combined.

EDYou

The college admissions process is already daunting enough for any college applicant, but it can be even more difficult if you’re a first-generation student. EDYou is designed to be an admissions guide – a companion for walking you through the admissions process, providing both personalized and general task lists, deadlines, financial aid guides, and scholarship finders for any college applicant.
